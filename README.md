# LeetCode Solutions

This repository contains my LeetCode practice solutions, organized by problem number and title.

## Repository Structure

- Each problem has its own folder named as:
  - `<problem-number>-<problem-title>`
- Inside each folder:
  - `README.md` – problem statement and details
  - Solution file (`.cpp` or `.sql`, depending on the problem)

## Current Progress

- **58** problem folders
- Solutions primarily in **C++** and **SQL**

## How to Use

1. Open any problem folder.
2. Read the local `README.md` for the problem statement.
3. Review the corresponding solution file.
